# Nexus Dotfiles

A customized Linux dotfiles repository for my Arch Linux VM/desktop setup.

## About
This repository contains configuration files, scripts, and desktop setup resources for a Hyprland/Sway based Linux workflow. It is organized for easy installation, backup, restore, and future maintenance.

## Included Components
- `eww`
- `hypr`
- `sway`
- `waybar`
- `wofi`
- GTK theme and system design assets
- Installation, backup, and restore scripts
- Project documentation and record material

## Features
- Clean folder structure.
- Easy backup and restore workflow.
- Compatible with an Arch Linux based environment.
- Ready for academic project submission and personal use.

## Installation
```bash
chmod +x scripts/*.sh
bash scripts/install.sh
```

## Backup
```bash
bash scripts/backup.sh
```

## Restore
```bash
bash scripts/restore.sh
```
